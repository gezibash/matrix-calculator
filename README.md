## Features

- Neon green Matrix color scheme
- Animated falling character rain (katakana, numbers, symbols)
- CRT scanline effects
- Glowing buttons with hover animations
- Text flicker effect on display
- Fully functional calculator

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn/ui components

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to enter the Matrix.
